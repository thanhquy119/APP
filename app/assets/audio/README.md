# Focus Audio Assets

Muc nay chua cac file am thanh nen cho tinh nang "am thanh ho tro tap trung".

## Mapping trong app

- `Mua nhe` -> `mixkit-light-rain-loop-2393.wav`
- `Rung` -> `mixkit-night-crickets-near-the-swamp-1782.wav`
- `Suoi` -> `mixkit-birds-chirping-near-the-river-2473.wav`
- `White noise` -> `white_noise.wav`
- `Pink noise` -> `pink_noise.wav`
- `Brown noise` -> `brown_noise.wav`

## Ghi chu

- Neu 3 file noise (`white_noise.wav`, `pink_noise.wav`, `brown_noise.wav`) chua ton tai,
  app se tu sinh file noise ngan de loop.
- Co the thay file bang ban thu am khac voi cung ten de tuy bien chat am.
