# Assets folder for FocusGuardian

## Structure:
- `nature/` - Nature sounds for relaxation
- `icon.ico` - Application icon (optional)
- `sounds/` - Notification sounds (optional)

## Creating icon.ico

You can create an icon using online tools:
1. https://icoconvert.com/
2. https://convertio.co/png-ico/

Recommended sizes: 16x16, 32x32, 48x48, 256x256
