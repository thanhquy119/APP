# Nature Sounds Assets

This folder contains nature sounds for relaxation during breaks.

## Suggested sounds:
- rain.wav - Rain sounds
- forest.wav - Forest ambience
- waves.wav - Ocean waves
- birds.wav - Bird chirping

## How to add sounds:
1. Download or record nature sounds in WAV or MP3 format
2. Place them in this folder
3. Configure in Settings > Notifications > Sound file

## Free sound resources:
- https://freesound.org/
- https://www.zapsplat.com/
- https://mixkit.co/free-sound-effects/

Note: Keep files under 5MB each for optimal performance.
